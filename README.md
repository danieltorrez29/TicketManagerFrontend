## Getting Started

### Prerequisites

- Node.js
- npm

## Usage

### Localhost

Install the dependencies from the root folder on each micro frontend using the following command:

```
npm install
```

Then execute:

```
npm start
```

### Web

Click on the [[link]](https://www.netlify.com) to access the application deployed in Netlify.
